# Enterprise DevOps Platform

A full-stack cloud deployment platform inspired by Vercel, Railway, and Render.

## Features

- Real-time deployment logs
- GitHub repository deployments
- BullMQ background workers
- Redis queue system
- Docker container deployments
- PostgreSQL database
- Prisma ORM
- Socket.IO real-time updates
- Deployment monitoring dashboard
- Live deployment status tracking

## Tech Stack

### Frontend
- React
- Vite
- TailwindCSS
- Axios
- Socket.IO Client

### Backend
- Node.js
- Express.js
- Prisma ORM
- PostgreSQL
- Redis
- BullMQ
- Dockerode
- Socket.IO

## Architecture

- Frontend Dashboard
- REST API Backend
- Redis Queue Workers
- Docker Deployment Engine
- PostgreSQL Persistence Layer
- Real-time Socket.IO Updates
